[はじめに]
こちらの作品を閲覧していただきありがとうございます。
この作品はHAL東京就職作品プレゼンにて作成した作品になります。
詳しい説明は私の自己紹介サイトの作品ページで閲覧できます。


[概要]
自転車にGPSモジュール、気圧センサー、加速度センサー、磁気センサーを取り付けたラズパイを載せて走り、
得たデータをスプレッドシートへ保存。
スプレッドシートのデータをWEBサイトで閲覧できるシステムになります。

[使用したもの]
・Raspberry Pi zero W
・GPSモジュール
・気圧センサー
・加速度センサー
・磁気センサー

（環境により変わる物）
i2c接続に使用したもの
・SparkFun Qwiic SHIM for Raspberrypi
・Qwiic Cable x3
・モバイルバッテリー
・デザリングできる環境
